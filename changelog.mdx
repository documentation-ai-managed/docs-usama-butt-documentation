<Update label="2024-10-15" description="v1.2.0" tags={["feature", "improvement"]}>

## New Features

- Added real-time collaborative editing for documentation pages
- Introduced advanced full-text search with syntax highlighting

## Improvements

- Optimized page load times by `>30%` using lazy loading
- Enhanced mobile responsiveness for better viewing on all devices

## Bug Fixes

- Fixed rendering issues with nested code blocks
- Resolved authentication errors during session timeouts

</Update>

<Update label="2024-09-20" description="v1.1.0" tags={["feature", "bugfix"]}>

## New Features

- Integrated custom MDX components like `<Steps>` and `<Tabs>`
- Added brand color theming with `#3B82F6` as default blue

- Streamlined navigation with auto-generated sidebar
- Better accessibility support including ARIA labels

## Bug Fixes

- Corrected parsing errors for special characters like `{` and `<`

</Update>

## How to Upgrade

Follow these steps to upgrade your Usama Butt Documentation instance to the latest version.

<Steps>
  <Step title="Backup Your Site" icon="database">
    Create a full backup of your repository.
  </Step>
  <Step title="Pull Latest Changes" icon="git-pull-request">
    
    ```bash
    git pull origin main
    ```
  </Step>
  <Step title="Rebuild and Deploy" icon="rocket">
    
    ```bash
    npm run build
    npm run deploy
    ```
  </Step>
</Steps>
